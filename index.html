Definition

  • Hypertext Preprocessor
  • un langage de programmation libre
  • principalement utilisé pour produire des pages Web dynamiques
  • fonctionne comme n'importe quel langage interprété de façon locale

HISTOIRE

  • Créée en 1994
  • par Rasmus Lerdorf
  • Réécrit 3 fois
  • Mai 1998 :
    • 60.000 domaines exportaient l'en-tête "PHP"
    • 1% des domaines d'Internet de l'époque

Les conditions

                                $boolean = true;
$boolean2 = false;

if ($booleen2) {
  print 'Je suis le boolean 2 !';

  • Opérateurs if, elseif, else
  • Code entouré par des accolades (brackets)
  • Traite la condition avant d’exécuter le code

la boucle for

  • Opérateur For
  • Effectue le nombre de fois la boucle selon la validité de la condition
                                

for ($i = 0; $i < 10; $i++) {
  print "Boucle numéro " + $i;
}

la boucle while

  • Opérateur While
  • Effectue le nombre de fois la boucle selon la validité de la condition
                                
$i = 10;
while ($i > 1) {
    print 'Boucle numéro : ' + $i;
    $i--;
}

la boucle Do while

  • Opérateur Foreach
  • Effectue une fois la boucle et itère en fonction de la validité de la condition
  • Point virgule après la définition de la boucle !
                                
$i = 10;
do {
    print 'Boucle numéro : ' + $i;
    $i--;
} while ($i > 1);

la boucle Do while

  • Opérateur Foreach
  • Itère sur l'ensemble des éléments d'un tableau ou d'un objet
                                
$tableau = ["un", "deux", "trois", "quatre"];
$objet = {
    "un": "toto",
    "deux": "tata",
    "trois": "tutu"
};

for (var index in $tableau) {
    print "Je suis l'index " + index + " du tableau";
}

for (var index in $objet) {
    print "Je suis l'index " + index + "de l'objet";
}

les fonctions

  • Commence par le mot `function`
  • Possède un nom unique
  • Permet de séparer le code pour le réutiliser à différent endroit
                                
function auCarre(a) {
    return a * a;
}

$nombre = 5;

print auCarre($nombre); // retourne 25
print auCarre($nombre + 1); // retourne 36

INCLUre des fichiers php

  • Fonction `require`
  • Inclusion multiple d'un même fichier
  • Si le fichier n'est pas disponible, le script plante
                                <?php
    require 'header.php';
?>

<p>Ceci est mon contenu</p>

    require 'footer.php';
?>

INCLUre des fichiers php

  • Fonction `include_once`
  • Inclusion unique d'un même fichier
  • Si le fichier n'est pas disponible, le script continue
  • Si le fichier a déjà été inséré, il n'est pas réinséré
                                <?php
    include_once 'header.php';
?>

<p>Ceci est mon contenu</p>

<?php
    include_once 'header.php'; // n'affiche rien
?>

le formulaire

requete http

  • Protocole de communication entre les serveurs
  • 2 méthodes :
    • GET : Via l'URL
    • POST : Via la requête HTTP
  • Comporte un entête et un corps
